## qBittorent

Install [qBittorrent](https://www.qbittorrent.org/) on macos:

```bash
brew install tbo47/tap/qbittorrent
```

Then open it:

```bash
open -a qBittorrent
```

## aMule

Install [amule](https://amule-org.github.io/) on macos:

```bash
brew install tbo47/tap/amule
```

Then open it:

```bash
open -a aMule
```

Where to go next? [amule-discoveries](https://github.com/tbo47/amule-discoveries) or [amutorrent](https://github.com/got3nks/amutorrent) maybe.

Note: I tried to push it to the main brew repo but got this error: `The homebrew/cask tap requires all casks to be signed and notarized by Apple.`

Note 2: [tgragnato](https://github.com/tgragnato/homebrew-tap) also has a amule cask with some modifications to the amule code. Unlike this cask which has no modifications to the official amule software.
